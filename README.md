# FuelD Handle Website

Static site for FuelD Plaid compliance and OAuth handling.

## Cloudflare Pages

Use Git integration with these settings:

- Production branch: `main`
- Framework preset: `None`
- Build command: `exit 0`
- Build output directory: `/`
- Root directory: leave blank

Custom domain targets:

- `https://fueldapp.ai/`
- `https://fueldapp.ai/privacy/`
- `https://fueldapp.ai/plaid/`
- `https://fueldapp.ai/.well-known/apple-app-site-association`
